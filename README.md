# FlyVim

#### Quick Install

```bash
$ bash <(curl -fsSL https://raw.githubusercontent.com/if2012/FlyVim/master/install.sh)
```
#### Manual

```bash
$ git clone https://github.com/if2012/FlyVim.git ~/.FlyVim
$ cd ~/.FlyVim
$ make vim     # install FlyVim for Vim
$ make neovim  # install FlyVim for NeoVim
```
