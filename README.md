# FlyVim

## Install

#### Quick Install

```bash
$ bash <(curl -fsSL https://git.io/fNg5a)
```
#### Manual

```bash
$ git clone https://github.com/if2012/FlyVim ~/.FlyVim
$ cd ~/.FlyVim
$ make vim     # install FlyVim for Vim
$ make neovim  # install FlyVim for NeoVim
```
